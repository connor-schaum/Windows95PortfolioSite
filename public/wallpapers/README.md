# Wallpapers Directory

Add your wallpaper images here!

## Recommended Format
- **JPEG/JPG** - Best for photos, smaller file sizes
- **PNG** - Works but larger file sizes
- **WebP** - Modern format, great compression

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## How to Use
1. Add your wallpaper image files to this folder
2. Update `src/components/Settings.js` to include your wallpaper in the list
3. Or use the filename directly: `/wallpapers/your-image.jpg`

## Example
If you add `sunset.jpg` to this folder, you can reference it as:
- URL: `/wallpapers/sunset.jpg`
- Or add it to the wallpapers array in Settings.js
